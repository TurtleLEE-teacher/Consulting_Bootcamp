# 컨설팅펌 직무부트캠프 강의자료

컨설팅 산업과 A-E-D-B-D 방법론을 다루는 2~2.5시간 분량의 강의 슬라이드 프로젝트입니다.

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **강의 시간** | 2시간 ~ 2시간 30분 + Q&A 1시간 |
| **대상** | 컨설팅펌 취업 준비생 |
| **산출물** | PDF 슬라이드 (Reveal.js 기반) |

## 기술 스택

```
Reveal.js    - HTML 프레젠테이션 프레임워크
Mermaid.js   - 다이어그램/플로우차트
Pretendard   - 한글 웹폰트
```

## 폴더 구조

```
Consulting_Bootcamp/
├── README.md           # 프로젝트 개요 (현재 파일)
├── PRD.md              # 설계 명세 및 진행 상태
├── CLAUDE.md           # 개발 가이드 및 톤앤매너
├── sessions/           # 원본 강의 자료 (MD 파일)
├── slides/             # Reveal.js 슬라이드 소스
│   ├── index.html      # 메인 슬라이드 파일
│   ├── css/            # 커스텀 스타일
│   └── assets/         # 이미지, 아이콘
└── output/             # 최종 산출물 (PDF)
```

## 개발 방식

### 1. Reveal.js 선택 이유
- 고품질 시각화 (SVG, Mermaid 다이어그램)
- PDF 변환 공식 지원 (`?print-pdf`)
- 마크다운 기반으로 수정 용이
- 한 슬라이드 = 한 페이지 보장

### 2. PDF 변환 프로세스
```bash
# 1. 로컬 서버 실행
npx serve slides/

# 2. 브라우저에서 열기
http://localhost:3000?print-pdf

# 3. Ctrl+P → PDF로 저장
# - 배경 그래픽 체크
# - 여백: 없음
# - 용지: A4 또는 Letter
```

### 3. 슬라이드 설계 원칙
- 슬라이드당 핵심 메시지 1개
- 글머리 기호 최대 5개
- 다이어그램 우선, 텍스트 최소화
- 일관된 색상 팔레트 유지

## 강의 구성

### Part 1: 컨설팅과 컨설턴트 (60분)
- 컨설팅 개요 및 산업 구조
- PI vs BPR 비교
- A-E-D-B-D 방법론 소개
- 5C 역량 모델
- 커리어 패스 & Exit 옵션

### Part 2: A-E-D-B-D 방법론 (60~90분)
- ASSESS: As-Is 분석과 이슈 도출 (20분)
- ENVISION: 개선 기회와 To-Be 방향 (18분)
- DESIGN: 프로세스 혁신 설계 (22분)
- BUILD: 시스템 구현 핵심 (7분)
- DEPLOY: Go-Live와 안정화 (5분)
